# Addressable LED Controller

## 📌 Overview

This project provides a simple and extensible implementation for controlling addressable LED strips (e.g., WS2812 / RGB LEDs) using a microcontroller environment.

The goal is to enable dynamic lighting effects, color manipulation, and customizable LED behavior through programmable logic.

---

## 🚀 Features

* Control individual LEDs (addressable control)
* Support for RGB color manipulation
* Custom lighting effects and animations
* Easy to extend for new patterns and behaviors
* Lightweight and hardware-oriented implementation

## 🔄 How It Works

1. Initialize LED strip and define number of LEDs
2. Send data signals to control each LED individually
3. Apply color transformations (RGB values)
4. Render animations or static patterns

---

## ▶️ Getting Started

### Requirements

* Microcontroller (Arduino, ESP32, etc.)
* Addressable LED strip (WS2812 or similar)
* Development environment (Arduino IDE / PlatformIO)

### Installation

1. Clone this repository:

```bash
git clone https://github.com/Cijas/adressable_led.git
```

2. Open the project in your preferred IDE

3. Upload the code to your microcontroller
